# DocuDeskew

Устранение перекоса сканированных документов.  
Часть проекта **DocuMind** — интеллектуальной обработки юридических документов.

## Что делает

Принимает на вход grayscale-изображение страницы с белым фоном, находит границы документа, вычисляет угол наклона и выпрямляет его так, чтобы строки текста стали горизонтальными, а края — параллельными рамке. Не изменяет читаемую ориентацию (верх/низ) — это следующий этап `DocuOrient`.

## Возможности

- Работает только с 8-битными grayscale-изображениями (PNG/JPEG).
- Обнаруживает документ на белом фоне любой яркости.
- Определяет угол перекоса в диапазоне приблизительно от –45° до +45°.
- Выпрямляет изображение, обрезает пустые поля и добавляет настраиваемый отступ.
- Возвращает угол, ориентацию (портрет/ландшафт) и готовый буфер изображения.

## Алгоритм (кратко)

1. Детектор краёв (свёртка Собеля) + бинаризация.
2. Морфологическое усиление контура документа (дилатация + эрозия).
3. Сбор координат пикселей контура.
4. Итеративный поиск минимального ограничивающего прямоугольника (minAreaRect).
5. Аффинный поворот через `sharp` + обрезка + добавление padding.

## Установка

```bash
npm install docu-deskew
```

## API

```js
const { deskew } = require('docu-deskew');

const result = await deskew(imageBuffer, options);
// {
//   angle: number,          // угол перекоса в градусах
//   orientation: "portrait" | "landscape",
//   deskewedImage: Buffer   // выровненное grayscale PNG
// }
```

### Параметры options (необязательные)

| Параметр | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `edgeThreshold` | `number` | `25` | Порог бинаризации градиента (0–255) |
| `dilateIterations` | `number` | `2` | Итераций дилатации |
| `erodeIterations` | `number` | `2` | Итераций эрозии |
| `padding` | `number` | `10` | Отступ после обрезки (px) |

## Пример использования

```js
const fs = require('fs');
const { deskew } = require('docu-deskew');

(async () => {
  const input = fs.readFileSync('scan.png');
  const { deskewedImage, angle } = await deskew(input);
  console.log(`Перекос: ${angle.toFixed(2)}°`);
  fs.writeFileSync('straight.png', deskewedImage);
})();
```
